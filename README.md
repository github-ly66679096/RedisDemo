# RedisDemo
这个是一个用来练习使用redis的demo。 程序运行首先通过ajax的方式获取mysql或者直接读取redis中的数据，然后以json数据的方式传给jsp， 然后在jsp中直接对得到的josn数据进行解析。并直接显示省份和城市的数据。

本demo非常简单：

首先创建工程---完成静态界面和MVC框架
在mysql中创建好过程中要使用的数据；
准备好过程中要使用的各种jar或者直接创建一个Maven工程，导入依赖；
代码编写，完成需要。
